关于类型转换与精度

CASE(value AS type)

ROUND(value,精度)

CONVERT（value，type）

type:

  • 二进制，同带binary前缀的效果 : BINARY
  • 字符型，可带参数 : CHAR()
  • 日期 : DATE
  • 时间: TIME
  • 日期时间型 : DATETIME
  • 浮点数 : DECIMAL
  • 整数:SIGNED
  • 无符号整数 : UNSIGNED

FORMAT（X,D）：强制保留D位小数，整数部分超过三位的时候以逗号分割，并且返回的结果是string类型的
